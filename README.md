##项目介绍

> 本项目主要用于熟悉react以及react技术栈

> 项目内容为一个做测试题的app，请用chrome的手机模式浏览哦。

> 技术栈：react+react-route+redux+webpack

## 运行项目

```bash
# 克隆项目
git clone https://github.com/gitgit-suzume/react-test.git

#进入项目文件夹
cd react-test

#下载依赖包
npm i

#运行项目(端口为8800)
npm start
```
